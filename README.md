# Andavi
International Price Alerts Platform
Frontend:
  - Next.js 14+ (App Router)
  - React 18
  - TailwindCSS
  - Chart.js (gráficos de precios)

Backend:
  - Next.js API Routes
  - Node.js runtime

Database:
  - PostgreSQL (recomendado)
  - Prisma ORM

Autenticación:
  - NextAuth.js
  - Google OAuth

Pagos:
  - Stripe (suscripciones)

Email:
  - SendGrid / Resend

Scraping:
  - Axios + Cheerio
  - Cheerio para parsing HTML

Hosting:
  - Vercel (recomendado)
  - Railway (alternativa)

Cron Jobs:
  - Vercel Cron Jobs
